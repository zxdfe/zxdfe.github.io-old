Charles报错处理

Charles报错Failed to install helper CFErrorDomainLaunchd error 9

  • 打开终端输入launchctl print-disabled system,回车查看com.xk72.charles.ProxyHelper是否为true
  • 如果是的话,输入sudo launchctl enable system/com.xk72.charles.ProxyHelper回车,输入系统密码
  • 重启Charles

Mac 本地Https抓包配置

  1. 安装根证书 Help -> SSL Proxying -> Install Charles Root Certificate

  2. 钥匙串中点击该证书,始终信任

  3. 设置HTTPS端口 Proxy -> SSL Proxying Settings 点击Enable SSL Proxying,点击Add

  4. 设置抓包地址和IP

手机上Https抓包配置

  1. 首先要保证和Mac在同一wifi下,查看Mac本地wifi的IP ifconfig | grep "inet" 或者wifi那里查看
  2. 手机上无线局域网点击该wifi,底部配置代理,选择手动配置 服务器填PC端的IP, 端口默认8888
  3. PC上Proxy -> Proxy Settings,同样设置8888
  4. 手机浏览器访问chls.pro/ssl, 此时电脑上提示,选择Allow允许
  5. 下载证书,并信任证书
  6. 同样设置需要抓包的HTTPS地址,或者*全局
